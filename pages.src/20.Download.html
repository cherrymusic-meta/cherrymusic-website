Arch quick install

This quick guide assumes that you use yaourt for AUR package installation.

Just download cherrymusic straight from the AUR:

Minimal Installation (no optional dependencies)

yaourt -S cherrymusic

Full Installation (all optional dependencies)

yaourt -S cherrymusic lame vorbis-tools flac faad2 mpg123 ffmpeg python3-stagger-svn imagemagick
