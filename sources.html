Use the source, Luke

  1. We're on github. Checkout or download the cherrymusic sources, e.g.:

    $ git clone git://github.com/devsnd/cherrymusic.git

  2. To use cherrymusic you need CherryPy. There are two options:

    • If you don't want to install it: CherryMusic can keep a local copy of the module for you, so you don't have to care about it.

    • Install it. Use your favorite package manager, or download it and do a manual install.

    cherrymusic will offer to do this for you if it cannot find an installed CherryPy to use.

  3. Now simply run the main script using python 3; On first start CherryMusic will create a config file for you, that you need to edit.

    $ python cherrymusic

  4. Open your browser and play some music!

    $ firefox localhost:8080
